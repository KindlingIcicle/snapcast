## Usage

> Cast your screen in a snap. Snapcast is a simple-to-use, responsive screencast app.

## Requirements

- Node 0.12.0
- Express 4.13.3
- Angular 1.4.7
- Mongoose 4.2.5
- Socket.io 1.3.7
- Grunt-Docco 0.4.0
- Body-Parser 1.14.1
- GetScreenMedia: 1.5.3
- GetUserMedia: 1.3.5

## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

### Packaging Documentation

From within the root directory:

```sh
grunt docco
```
